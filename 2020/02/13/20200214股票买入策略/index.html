1、股票的海选

首先看一下巴菲特六个选股习惯
巴菲特选股非常注重风险管理，他的选股习惯值得我们借鉴和学习，让我们看看==巴菲特的六个选股习惯==。
1 大型股
2 长期稳定获利能力
3 高ROE，低负债，低市盈率
4 简单的企业
5 良好的经营团队
6 耐心等待买入时机

登录i问财首页，输入==连续5年roe大于20%，连续5年的净利润现金含量大于80%，连续5年毛利润大于30%==可以进行股票初筛，筛选出比较优质的股票，当然这里的连续几年可以设置低一些，可以看到更多优质的股票。

5、计算好价格


==合理价位区间的计算公式：合理股价 = 市盈率 x 净利率（去年） x 总资产周转率（去年） x 每股净值（当前）x 权益系数（去年）==
其中：每股净值 = 当前股价/当前市净率
以贵州茅台为例（通过雪球股票）



市盈率：33.4
市净率：10.91
净利润：51.37%
权益系数：1.36
每股净值 = 当前股价/当前市净率 = 1088/10.91 = 99.73
总资产周转率：0.5
则当前合理的价位为 = 33.4 x 51.37% x 0.5 x99.73 x1.36 == 1163.5元==
所以可以认为当前的价格是处于低估区间的。
但是如果按照市盈率在20左右为正常区间的话，则有
20 x 51.37% x 0.5 x99.73 x1.36 = ==696.7元==
但是评断一下在这个20是否合适，可以看整个行业的市盈率

行业平均市盈率为29.74
所以计算合理价格区间29.74 x 51.37% x 0.5 x99.73 x1.36 = ==1036元==
下面看一下财报说对贵州茅台的评级


可以看到计算的股价合理区间的差异不大。

6、制定买进标准

首先看韭菜的三个特点

第一个特点：韭菜是投机者而非投资者
投机者与投资者的区别：投资以赚现金流为主，赚差价为辅;投机以赚差价为主。
投资是赚确定性很现金流， 在现金收益率高的时候买入，比如投资者 会买进高股息率的好股票、高现金分红率的即使价格不涨甚至下跌，每年也有7%-15%现金分红收益。
而投机是通过预测价格变动来赚价差。这有很大的不确定性。因为价格的涨跌是无法预测的，投机其实就是赌博。历史上没有一位投资大师是靠预测价格涨跌变富有的。
生活中没有人是靠赌博成为富人的，但是有人靠开赌场成为了大富豪。为什么呢?因为赌徒是投机者，而开赌场的人是投资者。

第二个特点：追求短期收益，高频收益。
一、赢利的客户中，85.2%的赢利来自于5单以内的赢利，扣除掉这5单，这部分客户的大多数都是亏损。这5单的特点:持有日期基本上都在2个月以上。
二、==每日平均交易10次以上的客户，3年平均收益率是-79.2%。每日平均交易5次以上的客户，3年平均收益率是-55%，每日交易1次以上的客户，3年收益率是-31.5%，每日平均交易0.3次(3天- -次)以上的客户，3年平均收益率是12%;每日平均交易0.1次(10天- -次) 的客户，3年平均收益率是59%。==
三、所有止损的单，如果不止损，有98.8%的概率在未来2周内扭亏为赢。
所以结论是：1高频交易，基本上判了你被割韭菜的命运；2不要盼望你能在高抛低吸的短线交易中获胜。

第三个特点：不学习或者不深入学习，盲目乱投资
韭菜渴望通过投资快速暴富，但是不愿学习，这样一开始就注定了韭菜的命运。

制定买进标准

在合理或者被低估的价位区间，买入筛选出来的优质公司的股票。以贵州茅台为例，可以进行定投，目前价格属于合理的价位区间。
